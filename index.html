How "EOD Data for all DOW Jones Stocks" was processed

  1. All csv files were converted to json equivalents.
  2. Average time series file to represent index was created, ignoring DWDP index due to mismatched time domain.
  3. the following headers were dropped: ['unadjustedVolume', 'change', 'changePercent', 'label', 'changeOverTime'] from both files.
  4. File imported into linked javascript script
  5. In all selection based graphs, temporal data is either mapped to the earlier of the two selections, or to a neutral third time domain. This data is then fed to the graphs before displaying.
  6. Only for the case of Feb 2014, we fill the first 20 days (originally missing) with the value of Feb 21st 2014 for properly projecting data in all graphs.
**All visualizations currently based off of "close" price value.
